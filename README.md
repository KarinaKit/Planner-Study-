📅 Planner Study (Умный планировщик обучения)

Интерактивное веб-приложение для организации учебного процесса. Планировщик автоматически подстраивает список задач под выбранный день недели и позволяет отслеживать прогресс выполнения.

🚀 Основные функции:

Умный календарь: Интеграция библиотеки Flatpickr для удобного выбора даты с поддержкой русского языка.

Динамическое расписание: Автоматическая генерация списка занятий (Математика, Английский, TypeScript) на основе выбранного дня недели.

Система чекбоксов: Интерактивные задачи с визуальным подтверждением выполнения (зачеркивание текста).

Адаптивный дизайн: Использование медиа-запросов (@media) для корректного отображения на мобильных устройствах (от 320px до 480px).

Эстетика Glassmorphism: Современный интерфейс с полупрозрачными фонами и эффектом размытия (backdrop-filter).

🛠 Технологии:

HTML5:
Семантическая разметка, использование таблиц для расписания.


CSS3: 
Гибкая верстка и адаптивность.

Стилизация кастомных чекбоксов.

Эффекты прозрачности и размытия.


JavaScript (Vanilla JS):
Логика работы с объектом Date.

Делегирование событий для управления списком задач.

Динамическое обновление DOM.

Flatpickr: Внешняя библиотека для продвинутой работы с календарем.

Choices.js: Легкая библиотека на чистом JS для кастомизации элементов select
